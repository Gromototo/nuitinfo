Hierar

Project Date August, 2020
Languages Python, Js, SQL, HTML/CSS
Web Server Linux on Compute Engine
Framework and Proxy Flask, Bootstrap, SQlite and Nginx

Hierar allows the user to input their schedule, homework, estimated task duration, and availability for work. Using this data, the application automatically generates an optimized and detailed schedule, specifying which task should be prioritized.

Hierar also takes into account the results of supervised homework, as well as the specific coefficients for each subject. Additionally, it includes a notification feature that alerts the user when an excessive workload is detected in the future. This feature is a valuable aid in staying organized and optimizing time management, which is essential for academic success.

The main objective of Hierar is to assist its users in improving their academic performance.

Hierar is a web application that automatically generates a timetable to enhance the academic performance of its users based on their availability.
